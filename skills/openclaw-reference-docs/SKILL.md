# Obsidian Reference Lookup

skill นี้ไว้ให้ skill อื่นเรียกใช้เพื่อดึง context จาก Obsidian vault
ก่อนสรุปหรือตอบคำถามผู้ใช้

## วิธีใช้

เมื่อ skill อื่นต้องการ ref เพิ่มเติม ให้ทำตามขั้นตอน:

- [ ] **1.** ใช้ Obsidian MCP ค้นหาโน้ตที่เกี่ยวข้องกับ topic ปัจจุบัน
- [ ] **2.** อ่านโน้ตที่เจอ (ไม่เกิน 3 ไฟล์)
- [ ] **3.** นำ context ที่ได้ไปประกอบกับงานหลัก

> ✅ เมื่อทำเสร็จแต่ละขั้น ให้ตick [ ] ให้เป็น `[x]` — จะได้รู้ว่าขั้นไหนเสร็จแล้ว ขั้นไหนยังต้องทำ

## Output format

ส่งกลับเป็น:
📎 Obsidian Ref:
- [ชื่อโน้ต]: [สรุปสั้นๆ ที่เกี่ยวข้อง]
- [ชื่อโน้ต]: [สรุปสั้นๆ ที่เกี่ยวข้อง]

ถ้าไม่เจอ ref ที่เกี่ยวข้องให้ข้ามไปเลย ไม่ต้องแจ้ง
