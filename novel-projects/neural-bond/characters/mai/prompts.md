# ใหม่ (Mai) — Image Prompts

## 🎨 Base Prompt
```
19-year-old Thai male medical student, short black hair, warm brown eyes with curious expression, slightly above average height, lean build, warm skin tone, youthful face with soft features, wearing casual university outfit (white shirt + dark trousers), soft lighting, cinematic, contemporary Thai setting
```

## 😄 Emotion Variants

| Emotion | Prompt Addition |
|---------|----------------|
| **Curious** | `, curious expression, slightly tilted head, bright eyes` |
| **Shy** | `, shy smile, slight blush, eyes looking down` |
| **Focused** | `, determined look, furrowed brow, clinical focus` |
| **Sad** | `, melancholic eyes, slight frown, dim lighting` |
| **Happy** | `, genuine warm smile, crinkled eyes, bright natural light` |
| **Surprised** | `, wide eyes, slightly open mouth, taken aback` |
| **Flustered** | `, blushing, nervous smile, avoiding eye contact` |

## 👕 Outfit Variants

| Outfit | Prompt Addition |
|--------|----------------|
| **University** | `, white dress shirt, dark navy trousers, leather satchel bag, casual sneakers` |
| **Medical Scrubs** | `, light blue medical scrubs, stethoscope around neck, hospital setting` |
| **Casual Home** | `, oversized hoodie, messy hair, soft indoor lighting, cozy background` |
| **Night Scene** | `, dark hoodie, dimly lit corridor background, mysterious atmosphere` |

| **Lab scene** | `19-year-old Thai male medical student, wearing headphones, focused expression, dark underground lab background with blue monitor lights, sci-fi setting, dramatic lighting` |
| **Hospital clerk** | `Thai male medical student year 1, light blue scrubs, holding patient chart, standing in hospital corridor, professional yet nervous vibe, bright clinical lighting` |

## 📝 Prompt Writing Rules

- Always start with: age, ethnicity, gender
- Hair → eyes → face shape → body → outfit
- End with: lighting + vibe + quality tags
- Quality tags: `, highly detailed, cinematic lighting, 8k, sharp focus`
- Negative (if using): `, blurry, low quality, distorted face, extra limbs`
